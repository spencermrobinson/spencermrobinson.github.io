My Work

Robinson-Golf

VIEW GITHUB REPO VIEW LIVE SITE

Concept

I build Robinson-Golf with the dual purpose of providing customers with an easy to use e-commerce site and to allow business owners application that allows them to manage employees, inventory, orders and track sales.

Technology

Robinson-Golf was built in React using React-Redux. I used express, passport and auth0 to set up my server and manage authentication. I used massive to communicate between my server and my database with PostgreSql. I implemented Twilio to send customers a SMS text message when they completed their order and Nodemailer to send an email to the customer when the order was shipped. Stripe was used to process credit card orders.
